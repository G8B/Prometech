## Site

### Structure

Le site consiste en trois parties principales :
1. Interface client
    - Mon compte
    - Gestion logements
    - Tableau de bord
1. Interface gestionnaire d'immeuble
    - Mon compte
    - Gestion immeubles
    - Tableau de bord
1. Interface administrateur / Back office
    - Mon compte
    - Gestion produits
    - Gestion comptes
    - Support
    - Logs
